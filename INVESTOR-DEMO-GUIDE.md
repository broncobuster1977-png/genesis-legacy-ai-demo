## 🔧 TECHNICAL REQUIREMENTS

- **LiveKit Server:** Port 7880 (should be running)

- **Database:** PostgreSQL connection required
- **Network:** Ensure port 7880 is accessible

### Troubleshooting
- **Connection Issues:** Restart LiveKit server on port 7880
- **Agent Not Responding:** Check agent selection and scenario matching
- **Audio Problems:** Verify microphone permissions in browser
- **Video Issues:** Check camera permissions and bandwidth
