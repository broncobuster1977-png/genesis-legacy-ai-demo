- **Frontend:** Next.js 14 (App Router) + React 19 + TypeScript
- **Styling:** Tailwind CSS 4 + Custom Design System
- **Authentication:** NextAuth.js + JWT + Session Management
- **Database:** PostgreSQL + Prisma ORM + Encrypted Storage
- **AI Integration:** Anthropic Claude API + Streaming Responses
- **Voice:** LiveKit Client SDK + Browser Speech APIs
- **PWA:** Service Worker + Offline Caching + Push Notifications

- **Encryption:** AES-256-GCM for Soul Files and messages

- **Session Management:** HTTP-only cookies + CSRF protection
- **Input Validation:** Server-side validation and sanitization
- **Route Protection:** Middleware-based access control

## 🔧 IMMEDIATE SETUP REQUIRED

### **Environment Variables**
```bash
# Database
DATABASE_URL="postgresql://username:password@localhost:5432/legacyai"

# Authentication
NEXTAUTH_URL="http://localhost:3000"
NEXTAUTH_SECRET="your-secret-key-here"

# AI Integration
ANTHROPIC_API_KEY="sk-ant-your-key-here"

# Email (Optional for MVP)
EMAIL_SERVER_HOST="smtp.example.com"
EMAIL_SERVER_PORT=587
EMAIL_SERVER_USER="your-email@example.com"
EMAIL_SERVER_PASSWORD="your-password"
EMAIL_FROM="noreply@legacy-ai.com"
```

### **Database Setup**
```bash
# Install dependencies
npm install

# Generate Prisma client
npm run db:generate

# Set up database
npm run db:push

# Start development server
npm run dev
```
